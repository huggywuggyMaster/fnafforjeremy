In diesem Beispiel wird das Bild mit der ID "play-button" zunächst normal angezeigt. Dann wird in JavaScript ein Event-Listener hinzugefügt, der auf das "mouseover"-Event reagiert, wenn die Maus über das Bild schwebt. Im Event-Listener wird dann einfach die "display"-Eigenschaft des Bildes auf "none" gesetzt, um es auszublenden.
